# WPS格式规范分析技能

该技能用于分析WPS Excel文档中不同业务员创建的工作表格式规范，对比分析格式差异和业务逻辑差异，并生成标准化建议报告。

## 功能特点

1. **格式规范提取**：提取不同工作表的字体样式、段落间距、表格结构、页眉页脚、边框设置、单元格对齐方式等格式信息
2. **差异对比分析**：对比分析不同创作者之间在格式设置上的具体差异
3. **业务逻辑分析**：检查数据录入规则、公式计算逻辑、数据结构设计等业务逻辑差异
4. **标准化建议**：基于分析结果提出格式标准化和流程标准化的具体建议
5. **报告生成**：生成详细的差异分析报告，包含差异点标注和改进建议

## 使用场景

- 用户要求对比不同业务员文档格式差异
- 需要对现有文档进行标准化改造
- 分析不同创建者的数据录入规则差异
- 生成企业文档规范标准

## 分析维度

### 格式分析维度
- 字体样式（字体类型、字号、加粗、颜色）
- 单元格对齐方式（水平对齐、垂直对齐）
- 边框设置（边框样式、边框颜色）
- 背景填充（填充颜色、图案样式）
- 列宽行高设置
- 合并单元格使用情况
- 表头结构设计
- 条件格式使用
- 数据验证设置

### 业务逻辑分析维度
- 数据录入规则（空行处理、合计行位置、汇总方式）
- 公式计算逻辑（求和位置、汇总方式、引用方式）
- 表格结构设计（表头行数、数据排列方式）
- 日期格式规范
- 款号编码规则
- 颜色尺码命名规范
- 合计行处理方式

## 输出格式

生成的分析报告包含：
1. 文档基本信息概述
2. 格式差异点详细列表
3. 业务逻辑差异点详细列表
4. 综合分析结论
5. 标准化改进建议
