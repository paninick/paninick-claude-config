## 工厂结构

### 东川服装厂（本部）完整工序（18道）
工艺制版 → 横机 → 套口 → 服装打样 → 制版 → 裁剪 → 验片 → 套口 → 检验 → 照灯 → 水洗 → 锁扣 → 钉标 → 缝制 → 整烫 → 检验 → 挂牌 → 检针 → 包装装箱

### 沭阳毛纺厂（外协）工序（13道）
横机 → 套口 → 检验 → 照灯 → 水洗 → 锁扣 → 钉标 → 缝制 → 整烫 → 检验 → 挂牌 → 检针 → 包装装箱

### 外加工（第三方）
按实际委托工序记录，需指定工序范围和交接节点

## 设计规则

1. 每道工序必须有：工序编码、工序名称、所属工厂、前置工序、后置工序、负责岗位、质检节点标记
2. 质检节点（检验、照灯）必须有不合格品处理流程（返工/报废/降级）
3. 外发工序需记录：外发单位、外发数量、预计回厂日期、实际回厂数量、损耗数量
4. 损耗管控：每道工序设定损耗上限，超出需审批
5. 工序流转状态：待开工 / 进行中 / 完工待检 / 检验通过 / 检验不合格 / 外发中

## 款号主键规则
格式：KN-{年份}-{季节码}-{流水号}-{客户码}-{版本号}
示例：KN-2026-S-001-A-02

## 工序间交接验收标准（放行门槛，不达标不流转）

| 交接节点 | 放行条件 |
|----------|----------|
| 染整 → 裁剪 | 缩水率偏差 ≤±0.5%、色差 ≥4级、纬斜 ≤3% |
| 裁剪 → 拼接 | 裁片尺寸偏差 ≤±2mm、缸号标识完整、验片合格章 |
| 拼接 → 缝制 | 拼缝强度测试通过、无开缝脱线、缝份均匀 |
| 缝制 → 整烫 | 线头清理完毕、套结点位合格、尺寸复核通过 |
| 整烫 → 检验 | 定型尺寸符合工艺单、无极光烫痕、无变形 |
| 检验 → 包装 | 外观全检通过、检针通过、挂牌标签完整 |

不合格处理：返工（可修复）/ 降级处理（不影响功能性缺陷）/ 报废（不可修复）

## 外发路由决策规则

| 条件 | 路由 |
|------|------|
| 横机产能利用率 ≥85% | 自动路由沭阳毛纺厂 |
| 拼接工序（异面料） | 仅走东川本部，禁止外发 |
| 普通缝制/整烫产能不足 | 可路由外加工，需质量门槛审批 |
| 外加工回厂验收不合格率 ≥5% | 暂停该外加工单位路由 |

外加工回厂必须：数量核对 + 质检抽检（抽检比例 ≥10%）+ 损耗确认

## 优化落地优先级

| 优先级 | 模块 | 对应章节 |
|--------|------|----------|
| P0（上线前必须） | 缺陷闭环管控、JIS合规验厂 | 第九、十一章 |
| P1（上线后首批） | 拼接工艺SOP、染整阈值约束 | 第五、三章 |
| P2（运营优化期） | 柔性排单、全链路损耗管控 | 第七、八章 |
| P3（长期战略） | 品类拓展、供应链升级 | 第十四章 |

## 输出物要求
- 工序流转设计必须包含：流程图描述 + 数据库表结构 + 状态机定义
- 外发设计必须包含：外发单模板字段 + 回厂核对规则 + 路由决策逻辑
- 质检节点设计必须包含：放行标准 + 不合格处理流程
